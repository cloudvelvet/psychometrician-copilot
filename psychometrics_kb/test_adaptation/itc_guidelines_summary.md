## Summary

Adapting a test is not just translating item text. The workflow should preserve construct meaning, language clarity, administration comparability, response process, and score interpretation in the target population.

## Checks

- Document translation, expert review, pilot testing, and item revision decisions.
- Check whether the construct and item content carry the same meaning in the target context.
- Use invariance and DIF analyses as evidence, not as substitutes for cultural review.

## Use When

- The user mentions translation, localization, or use in a new culture/language.
- A scale is imported from another population or setting.
